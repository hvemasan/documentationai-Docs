---
title: File Upload Error
description: Quick fix if your resume won't upload.
---

## Symptoms

- Error: "Invalid file format" or "Upload failed"
- File not processing

## Quick Fix

<Steps>
  <Step title="Check file type">
    Ensure your resume is PDF or DOCX.
  </Step>
  <Step title="Check file size">
    File must be under 5MB.
  </Step>
  <Step title="Try renaming file">
    Use standard letters/numbers, with no special characters.
  </Step>
  <Step title="Refresh and retry">
    Reload dashboard and upload again.
  </Step>
</Steps>

<Callout kind="info">
Still having issues? Email support@resumeup.ai with your file.
</Callout>
